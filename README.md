# Analisador de Gastos Pessoais

Projeto em Python que analisa gastos mensais por categoria, 
calcula totais e identifica o maior gasto do período.

## Funcionalidades
- Total geral do mês
- Gastos agrupados por categoria com percentual
- Identificação do maior gasto

## Tecnologias
- Python 3
- Estruturas de dados nativas (listas e dicionários)

## Como executar
```bash
python3 main.py
```
